# hatenablog-thought

[はてなブログ](https://hatenablog.com/)のブログテーマ「Thought」です。

テーマストアから[インストールして使う](http://blog.hatena.ne.jp/-/store/theme/17680117126965268278)ことができます。

## 開発

セットアップ：

```
yarn install
```

Sassの監視とコンパイルを開始：

```
gulp
```

はてなブログの「デザインCSS」に設定する：

```
@import 'http://localhost:3030/thought.css';

```
